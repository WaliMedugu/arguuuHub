Framework

State Management

A reactive caching and state-management framework for Flutter, ensuring performance and testability.

Supabase Flutter

Apache 2.0

Navigation

GoRouter

A declarative routing package for Flutter that uses the Router API to provide a convenient, url-based API.

MIT License

Typography

Google Fonts

A library of 1054 free licensed font families and APIs for conveniently using the fonts via CSS and Android.

OFL / Apache 2.0

Font Awesome Custom

Intl & Timeago

MIT / BSD
